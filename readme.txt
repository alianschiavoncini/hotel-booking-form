== Description ==
Hotel booking form displays a ready-to-use online booking form for accommodation facilities (hotels, b & b, hostels, motels, camping, etc ...). The user can fill it in to send the booking data to the email address of the site administrator.

The shortcode [hotel-booking-form] must be used to display the hotel booking form.

Form fields can be activated individually from the plugin settings page. The available form fields are:
- First name
- Last name
- Email
- Phone
- Check-in dates
- Departure dates
- Treatment options
- Room type options
- Number of rooms
- Number of adults
- Number of children
- Age of the children
- Message
- Privacy checkbox
- Title of the page
- URL of the page

Other options are available to the fields such as selecting the required fields to be filled in, customizing the treatment and room options, choosing the check-in and check-out date format, the calendar theme (jQuery datepicker is used) and more.

== Installation ==
1. Upload the entire `display-post-link` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
